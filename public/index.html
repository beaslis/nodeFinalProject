API Overview

Welcome to the API for managing users and cards. Below is a brief description of the available endpoints:

Users API

  • GET /users: Retrieve all users (admin only).
  • GET /users/:id: View details for a specific user.
  • POST /users/register: Register a new user.
  • POST /users/login: Login an existing user.
  • PUT /users/:id: Update user details.
  • DELETE /users/:id: Delete a user.

For more information, visit the Users Postman documentation.

Cards API

  • GET /cards: Retrieve all cards.
  • GET /cards/:id: View details of a specific card.
  • POST /cards: Create a new card (business only).
  • PUT /cards/:id: Update a card.
  • DELETE /cards/:id: Delete a card.
  • PATCH /cards/:id: Like or unlike a card.

For more information, visit the Cards Postman documentation.
